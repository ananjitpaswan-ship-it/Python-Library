<html><H1><B>My First Python Program: Hello, World!</B></H1>
<Br>
This repository contains my very first program written in Python. Following the common tradition for beginners, this script simply prints the message "Hello, World!" to the console.
<Br>
<H1>Purpose</H1>
<Br>
The main goal of this project is to serve as an initial exercise to:
Understand basic Python syntax.
Learn how to use the built-in print() function.
Confirm that a Python development environment is correctly installed and configured.
<Br>
<H1>Usage</H1>
<Br>
Ensure you have Python installed on your system.
Clone this repository to your local machine.
Navigate to the repository directory in your terminal or command prompt.
<Br>
Run the script using the following command:
<Table>
<tr>bash</tr>
<tr>python hello_world.py</tr>
</Table>
<H1>File</H1>
<Br>
hello_world.py: The single Python script file containing the "Hello, World!" program.
